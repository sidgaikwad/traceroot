### Install SDK

Then you can install the [TraceRoot.AI](http://traceroot.ai) SDK in your Python package by

```bash
pip install traceroot==0.0.5
```

## Configure

### Environment Setup

To run any of the examples in this folder, you can configure Traceroot using environment variables.

---

### Option 1: Create a `.env` file in the root of your project

```bash
# Open the .env file and update it with your actual Traceroot token and details
TRACEROOT_TOKEN=your-actual-traceroot-token
TRACEROOT_SERVICE_NAME=example
TRACEROOT_GITHUB_OWNER=traceroot-ai
TRACEROOT_GITHUB_REPO_NAME=traceroot
TRACEROOT_GITHUB_COMMIT_HASH=main
TRACEROOT_LOCAL_MODE=true
```

Then run your program:

```bash
python file_name.py
```

---

### Option 2: Manually export the environment variables

```bash
export TRACEROOT_TOKEN=your-actual-traceroot-token
export TRACEROOT_SERVICE_NAME=example
export TRACEROOT_GITHUB_OWNER=traceroot-ai
export TRACEROOT_GITHUB_REPO_NAME=traceroot
export TRACEROOT_GITHUB_COMMIT_HASH=main
export TRACEROOT_LOCAL_MODE=true
```

Then run your program:

```bash
python file_name.py
```

---

---

âœ… **Notes:**

- Only `TRACEROOT_TOKEN` and `TRACEROOT_SERVICE_NAME` are required.
- GitHub-related variables (`OWNER`, `REPO_NAME`, `COMMIT_HASH`) are optional.
- `TRACEROOT_LOCAL_MODE` can be set to `true` for local testing.

Then you can start tracing and logging by

```python
logger = traceroot.get_logger()

@traceroot.trace()
def main():
    logger.info("This is an info message")
```

And run your program:

```bash
python your_program.py
```

All of the logging and tracing will be automatically tracked and pushed to [TraceRoot.AI](http://traceroot.ai) (demo) platform.

### A Simple Example

[TraceRoot.AI](http://traceroot.ai) has been integrated with famous Python libraries like [camel](https://github.com/camel-ai/camel).

You can easily track the AI agent's tracing and logging by the following code:

```bash
git clone https://github.com/camel-ai/camel.git
cd camel
# Install camel

vim examples/debug/.traceroot-config.yaml
# Put your token to the file

python3 examples/debug/traceroot_example.py
```
